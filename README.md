# IID Workflows Map

Static GitHub Pages site and companion Markdown document for the IID construction automation workflows.

## Files

- `index.html` - interactive 1000-foot and drill-down workflow view.
- `IID_WORKFLOW_A_TO_Z.md` - comprehensive Markdown workflow document in ten stages.
- `styles.css` - responsive page styling.
- `script.js` - tab and drill-down behavior.

## Source Basis

This artifact was assembled from the local IID automation workspace on 2026-06-01. It summarizes the job-packet intake, construction-map extraction, SharePoint list/folder creation, scheduled dig-ticket coordinate updates, prefielding packets, tasking/pricing, purchase-order routing, and open post-PO workflow gaps.

Primary SharePoint site referenced:

```text
https://quantaservices.sharepoint.com/sites/IIDConstruction
```

The page avoids publishing fixed SharePoint totals because live data can change between automation runs.
